Installation

The recommended way to install the engine is to get it as an NPM package. In some cases it's not possible, so a browser-ready bundle is available for download and add directly to a html file.


Install as browser-ready bundle

If you cannot use NPM packages, you can download the browser-ready bundle from here and include it in your HTML file. 

          <script type="text/javascript" src="https://unpkg.com/@usonialab/sprotty-diagram/dist/sprotty.min.js"></script>

Install from NPM

The engine is available as an NPM package. The package itself is an ES6 module, so it can be used with several different bundlers.

You can install it by running the following command.

          npm install @usonialab/sprotty-diagram

After installation you can import it like any other packages.

          import { Sprotty } from '@usonialab/sprotty-diagram';
